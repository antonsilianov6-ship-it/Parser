# Вспомогательные скрипты

Эта папка содержит вспомогательные скрипты для настройки и обслуживания AllInclusiveParser.

## Скрипты

### auth_telegram.py
Скрипт для авторизации в Telegram API и сохранения сессии.

**Использование:**
```bash
python scripts/auth_telegram.py
```

**Когда использовать:**
- При первом запуске парсера
- При смене учетных данных Telegram
- При истечении сессии

### check_config.py
Скрипт для проверки конфигурации NotebookLM Automation.

**Использование:**
```bash
python scripts/check_config.py
```

**Что проверяет:**
- Наличие и корректность config.json
- Настройки NotebookLM (email, password)
- Настройки Automation (target_chat_id, расписание)
- Настройки Telegram API (API_ID, API_HASH)
- Наличие и корректность config/prompts.json

### fix_encoding.py
Утилита для исправления кодировки файлов проекта.

**Использование:**
```bash
python scripts/fix_encoding.py
```

**Что делает:**
- Сканирует все файлы проекта (.py, .txt, .md, .bat, .json)
- Определяет текущую кодировку
- Конвертирует файлы в UTF-8
- Добавляет команду `chcp 65001` в .bat файлы
- Создает лог в папке logs/

**Когда использовать:**
- При проблемах с отображением кириллицы
- После клонирования репозитория
- При работе на разных операционных системах

## Примечания

Все скрипты должны запускаться из корневой директории проекта:
```bash
# Правильно
python scripts/auth_telegram.py

# Неправильно
cd scripts
python auth_telegram.py
```

Скрипты автоматически добавляют корневую директорию в `sys.path`, поэтому импорты из `src/` работают корректно.
